核心模块一：单片机基础概念

定义：集成CPU、存储器、I/O接口于一体的微型计算机

特点：体积小、成本低、功耗低、可靠性高

应用领域：智能家居、工业控制、智能穿戴、汽车电子

学习提示：重点理解"单片"的含义——将计算机核心部件集成在一块芯片上

核心模块二：硬件结构与工作原理

基本组成（8051系列为例）

  • CPU：运算器+控制器，处理数据和指令
  • 存储器：
    • 程序存储器（ROM）：存放程序代码
    • 数据存储器（RAM）：临时存储数据
  • I/O接口：P0-P3四个并行接口，用于连接外设

引脚功能

引脚类型 作用
VCC/GND 电源正/负极
RST 复位引脚（高电平复位）
XTAL1/XTAL2 外接晶振，提供时钟信号
P3口 兼具通用I/O和特殊功能（中断、串口等）

核心模块三：指令系统与编程入门

数制基础

  • 二进制（0/1）、十六进制（0-F）与十进制转换
  • BCD码：用4位二进制表示1位十进制数

C语言编程基础

// 简单LED闪烁程序示例
#include <reg51.h>  
sbit LED = P1^0; // 定义LED连接在P1.0引脚

void delay(unsigned int t){  
    while(t--);  
}  

void main(){  
    while(1){  
        LED = 0; // 点亮LED  
        delay(50000); // 延时  
        LED = 1; // 熄灭LED  
        delay(50000);  
    }  
}
