单片机的内部结构

核心组件：

  • CPU：运算和控制中心
  • RAM：数据临时存储（128B-512B）
  • ROM：程序存储（4KB-64KB）
  • 并行IO口：P0-P3通用输入输出
  • 串行口：UART异步通信接口
  • 定时器/计数器：3个16位定时器（T0/T1/T2）
  • 中断系统：5个中断源（外部/定时/串口）
  • 时钟电路：11.0592MHz晶振
  • 控制信号：ALE、PSEN、EA等
  • 特色功能寄存器：特殊功能寄存器（SFR）

8051引脚功能

并行IO口：

  • P0：8位漏极开路输出，需外接上拉电阻
  • P1-P3：内部带上拉电阻，准双向IO口

电阻配置：

  • 上拉电阻：增强输出高电平驱动能力
  • 下拉电阻：稳定输入低电平状态

C51程序基础

数制转换：

  • 二进制（0b1010）→ 十进制（10）→ 十六进制（0x0A）

变量定义：

sbit LED = P1^0; // 位定义
unsigned char data cnt; // 数据区变量

基本语句：

  • 顺序结构、分支结构（if-else）、循环结构（for/while）

单片机最小系统电路

核心元件：

  • 单片机芯片（89C52）
  • 晶振电路（11.0592MHz + 22pF电容）
  • 复位电路（10K电阻 + 10uF电容）
  • 电源电路（5V直流供电）

并行IO的结构与应用

结构特点：

  • 准双向口设计，支持输入/输出切换
  • 每位可独立操作（位寻址功能）

典型应用：

  • 输出：LED流水灯、数码管显示
  • 输入：独立按键检测、矩阵键盘扫描
